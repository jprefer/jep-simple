# Simple - Jekyll Theme

A Jekyll version of the "Simple" theme 




You can use the following custom parameters in `_config.yml`.

## Social settings

- `500px_url`
- `facebook_url`
- `github_url`
- `gitlab_url`
- `googleplus_url`
- `instagram_url`
- `linkedin_url`
- `pinterest_url`
- `slack_url`
- `twitter_url`
